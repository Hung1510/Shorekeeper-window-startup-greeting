==============================================================
 HƯỚNG DẪN: PHÁT GIỌNG SHOREKEEPER KHI MỞ MÁY
==============================================================

Mục tiêu: khi đăng nhập vào máy tính, sẽ tự động phát giọng
nói của Shorekeeper.

Bạn cần 2 file:
  - shorekeeper_startup.exe   (chương trình phát tiếng)
  - shorekeeper_hello.wav     (file âm thanh)


--------------------------------------------------------------
 BƯỚC 1 - ĐỂ FILE VÀO ĐÚNG CHỖ
--------------------------------------------------------------

  1. Tạo thư mục tên  C:\Sounds
  2. Copy 2 file vào đó:
       - shorekeeper_startup.exe
       - shorekeeper_hello.wav
  3. Hai file phải nằm CHUNG một thư mục.


--------------------------------------------------------------
 BƯỚC 2 - THỬ NGHE TRƯỚC
--------------------------------------------------------------

  - Bấm đúp chuột vào  shorekeeper_startup.exe
  - Nếu nghe thấy tiếng -> OK, đúng rồi.
  - Lần đầu chạy có thể hiện bảng xanh "Windows protected
    your PC" -> bấm "More info" -> "Run anyway".


--------------------------------------------------------------
 BƯỚC 3 - HẸN CHO NÓ TỰ CHẠY KHI ĐĂNG NHẬP
--------------------------------------------------------------

  1. Bấm nút Start, gõ:  Task Scheduler  -> mở lên.
  2. Bên phải bấm:  Create Task...
  3. Thẻ GENERAL:
       - Name (tên): đặt gì cũng được, vd:  Shorekeeper
  4. Thẻ TRIGGERS -> bấm New...
       - "Begin the task" chọn:  At log on
       - Bấm OK
  5. Thẻ ACTIONS -> bấm New...
       - "Program/script" gõ:
         C:\Sounds\shorekeeper_startup.exe
       - Bấm OK
  6. Thẻ CONDITIONS:
       - BỎ chọn ô "Start the task only if the computer is
         on AC power"
  7. Bấm OK để lưu.

  Xong! Khởi động lại máy để thử.


--------------------------------------------------------------
 NẾU CÓ VẤN ĐỀ
--------------------------------------------------------------

  - Không nghe tiếng gì:
      File âm thanh phải tên đúng là  shorekeeper_hello.wav
      và nằm chung thư mục với file .exe.

  - Phát hơi trễ (vài giây sau khi mở máy):
      Đây là bình thường, không sao.

  - Thỉnh thoảng im lặng (không phát):
      Vào lại Task Scheduler -> mở task -> thẻ Triggers ->
      sửa trigger -> tick "Delay task for" -> chọn 2 seconds.


--------------------------------------------------------------
 MUỐN GỠ BỎ
--------------------------------------------------------------

  - Mở Task Scheduler -> chuột phải vào task -> Delete.
  - Xóa thư mục  C:\Sounds

  Vậy là sạch, không để lại gì.
